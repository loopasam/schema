Summary

Schema is a module for the Play! framework (1.2.X) which helps you to create full semantic stack web apps. The model and view layers of your Play! app are picked from a series of template classes following the schema.org structure. This module helps you to provide next generation content for search engines and to program your application in the scope of the Web.
